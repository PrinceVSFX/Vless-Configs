## О ПРОЕКТЕ &nbsp;+

Готовые конфиг-файлы для фильтрации контента.
Не нужно разбираться с настройками вручную —
всё уже организовано. Просто подключи и используй.

> Два файла. Один репозиторий. Полный контроль.

## КАК ИСПОЛЬЗОВАТЬ &nbsp;+

**Шаг 1** — Клонируй репозиторий
```bash
git clone https://github.com/PrinceVSFX/Adapt-Configs.git
```

**Шаг 2** — Перейди в папку
```bash
cd Adapt-Configs/Configs
```

**Шаг 3** — Подключи файлы к своему блокировщику
